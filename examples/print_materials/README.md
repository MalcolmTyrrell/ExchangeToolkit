# Example: print_materials
This example illustrates how to obtain all available color and material information from an input file. It reads the file specified on the command line and prints json-formatted output that reflects the color and material content.

The output by itself is not very useful, rather the point here is to demonstrate the process of extracting the available information for downstream visualization workflows.

## Sample Usage
`print_materials "_micro engine.CATProduct"`
